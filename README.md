# scarf :
Print a text scarf in various programming languages

## dependencies :
* Node.js LTS
* Python 3
* Clang-10
* Clang++-10
* mono C# compiler
* OpenJDK & JRE
* Ruby
* Crystal 0.27.2
* Golang
* unix-like Terminal Access (Linux, BSD, MacOS)

## running/compiling :
### C : main.c :
* `clang-10 -pthread -lm -o main main.c && ./main`

### C++ : main.cpp :
*  `clang++-10 -pthread -std=c++17 -o main main.cpp && ./main`

### C# : main.cs :
* `mcs -out:main.exe main.cs && mono main.exe`

### Python 3 : main.py :
* `python3 main.py`

### Node.js LTS : main.js :
* `node main.js`

### JSF*** : main.js :
* `node main.jsf`

### Java : Main.java :
* `javac Main.java && java Main`

### Ruby : main.rb :
* `ruby main.rb`

### Crystal : main.cr :
* `crystal main.cr`

### Golang : main.go :
* `go run main.go`
